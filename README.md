üìå Project Overview

This project focuses on analyzing beverage sales and profitability data for Coca-Cola retailers in 2021 using Microsoft Excel.
The objective is to transform raw sales data into actionable business insights by analyzing revenue, profit margins, regional performance, product trends, and seasonality.

The project demonstrates practical data cleaning, exploratory data analysis (EDA), pivot table analysis, and dashboard creation skills, which are essential for a Data Analyst role.

üéØ Business Objective

The primary goals of this analysis are:

To evaluate overall sales and operating profit performance

To identify top-performing beverage brands

To analyze regional and retailer-level contributions

To understand monthly and seasonal sales trends

To assess the relationship between sales volume and profitability

To support data-driven business decision-making

üìÅ Dataset Description

The dataset contains approximately 3,800 records of beverage sales transactions for the year 2021.

Key Attributes:

Retail Information: Retailer, Retailer ID

Geographical Data: Region, State, City

Time Data: Invoice Date, Month

Product Details: Beverage Brand

Sales Metrics: Price per Unit, Units Sold

Financial Metrics: Total Sales, Operating Profit, Operating Margin

This dataset represents a real-world FMCG sales scenario, making the analysis business-relevant.

üßπ Data Cleaning & Preparation

Before performing analysis, the following data preparation steps were completed in Excel:

Removed unnecessary header and blank rows

Renamed columns for clarity and consistency

Converted data types (dates, numeric values, percentages)

Created helper columns such as Month Name

Validated revenue calculations using price and units sold

These steps ensured the dataset was clean, accurate, and analysis-ready.

üí° Key Insights

Coca-Cola is the highest revenue-generating beverage brand
